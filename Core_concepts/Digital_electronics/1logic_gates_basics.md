# Logic Gates Basics

## Overview
Logic gates are fundamental building blocks of digital electronics.

## Concepts Covered
- AND gate
- OR gate
- NOT gate
- Truth tables
- Binary logic

## Key Equations
AND → Y = A · B

OR → Y = A + B

NOT → Y = Ā

## Automotive and Embedded Relevance
Logic gates are used in:
- ECU decision logic
- safety systems
- digital control systems
- sensor processing

## Example Application
Seat belt warning logic:
Warning = Ignition_ON AND Seatbelt_NOT_FASTENED

## Key Learning
Digital systems operate using binary logic and decision-making structures.
